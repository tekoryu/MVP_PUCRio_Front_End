Seja bem-vindo ao PDF Leser

Este é um protótipo de extrator de dados de PDFs, de forma a acelerar o trabalho em atividades onde os textos disponibilizados não estão estruturados. Novas funcionalidades serão incorporadas para o setor de Lobby na Atividade Legislativa, Crítica Literária e Ciência de Dados com foco em NLP. Fique com a gente, temos bolo!
